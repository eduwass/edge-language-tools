## Usage

```edge
@card()
  @slot('header')
    <h2>Card title</h2>
    <p>Card description</p>
  @end
  @slot('section')
    <p>Card content goes here.</p>
  @end
  @slot('footer')
    @button({ variant: 'outline' })
      Action
    @end
  @end
@end
```
