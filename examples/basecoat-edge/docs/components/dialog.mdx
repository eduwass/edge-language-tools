<Expandable title="Source">

```edge
@dialog({
  id: 'preview-dialog',
  trigger: 'Open dialog',
  title: 'Edit profile',
  description: 'Make changes to your profile here.',
})
  @slot('main')
    <p>Dialog body content.</p>
  @end
  @slot('footer')
    @button({ variant: 'outline' })
      Cancel
    @end
    @button()
      Save
    @end
  @end
@end
```
