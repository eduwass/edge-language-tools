## Usage

```edge
@dropdownMenu({
  id: 'preview-dropdown',
  items: [
    { label: 'Profile', type: 'item' },
    { label: 'Settings', type: 'item' },
    { type: 'separator' },
    { label: 'Logout', type: 'item' },
  ],
})
  @slot('trigger')
    Open menu
  @end
@end
```
