## Usage

```edge
@tabs({
  id: 'preview-tabs',
  tabsets: [
    { tab: 'Account', panel: '<p>Account settings</p>' },
    { tab: 'Password', panel: '<p>Change your password</p>' },
    { tab: 'Team', panel: '<p>Manage your team</p>' },
  ],
})
@end
```
