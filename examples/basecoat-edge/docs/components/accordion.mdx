## Usage

```edge
@accordion()
  <details open>
    <summary>
      Is it accessible?
      @svg('lucide:chevron-down')
    </summary>
    <section>Yes. It uses native disclosure semantics.</section>
  </details>
  <details>
    <summary>
      Is it styled?
      @svg('lucide:chevron-down')
    </summary>
    <section>Yes. It comes with basecoat styles.</section>
  </details>
@end
```
