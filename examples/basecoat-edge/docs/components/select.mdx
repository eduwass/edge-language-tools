## Usage

```edge
@select({
  id: 'preview-select',
  placeholder: 'Select a fruit',
  items: [
    { label: 'Apple', value: 'apple', type: 'item' },
    { label: 'Banana', value: 'banana', type: 'item' },
    { label: 'Orange', value: 'orange', type: 'item' },
  ],
})
@end
```
